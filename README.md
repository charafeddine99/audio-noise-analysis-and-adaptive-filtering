# Audio Noise Analysis and Adaptive Filtering

This project focuses on audio denoising using Digital Signal Processing (DSP) techniques.

## Features

- FFT Frequency Analysis
- Adaptive Filter Selection
- Low-pass Filter
- High-pass Filter
- Notch Filter
- Spectrogram Analysis
- MSE and SNR Evaluation

## Technologies

- Python
- NumPy
- SciPy
- Librosa
- Matplotlib

## Project Structure

```bash
dataset/
results/
main.py
requirements.txt
```

## Results

The system analyzes noisy audio signals and applies adaptive filtering techniques to reduce noise components.

## Metrics

- Mean Squared Error (MSE)
- Signal-to-Noise Ratio (SNR)
